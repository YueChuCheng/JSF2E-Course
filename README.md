## 課程大綱
>前端工程包含程式技術以及UI/UX設計兩個面向，在同學已經具有基本的技術以及設計能力的基礎上，本門課引導同學朝程式技術的進階邁進。課程內容包含JS、SCSS、webpack、Ajax、React、Redux、PWA。期待同學可以學習前端技術開發流程並學習更為完整的技術內容
